## Key Process Components

### Actors:
1. **User in Emergency:** Person needing first aid help
2. **Medaid Connect System:** Digital platform delivering instructions
3. **Database System:** Oracle database managing data and analytics

### Process Flow:
1. Emergency occurs → User opens app
2. User selects emergency type → System displays guide
3. User follows instructions → System tracks progress
4. User completes guide → Database logs statistics
5. User provides feedback → System updates analytics

### MIS Functions:
- Real-time data processing
- Decision support through analytics
- Performance monitoring of guides
- Resource optimization insights
